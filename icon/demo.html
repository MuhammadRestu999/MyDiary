How to use

It's very easy to use this IconFont, follow these steps:

Step 1 - Import IconFont, put this code inside <head>...</head> tag:

<link rel="stylesheet" href="/path/to/your/iconfont.css">

Step 2 - Choose the appropriate icon and get the class name, apply it to the page:

<i class="iconfont icon-xxx"></i>
